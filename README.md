# Paradise Nursery 🌿

## Project Overview

**Paradise Nursery** is a full-featured React-based e-commerce shopping application for plant lovers. Browse a curated collection of houseplants, add them to your cart, adjust quantities, and enjoy a seamless shopping experience built with modern React and Redux Toolkit.

---

## Project Name

**Paradise Nursery – Your Green Paradise Awaits**

---

## Features

- 🌱 Beautiful landing page with background image and "Get Started" button
- 🏪 Product listing page with 6+ plants organized into categories
- 🛒 Fully functional shopping cart with add, remove, and quantity controls
- 💳 Cart total and item count displayed in real time
- ⚡ State management powered by Redux Toolkit (CartSlice)
- 📱 Responsive design for desktop and mobile

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/paradise-nursery.git

# Navigate into the project
cd paradise-nursery

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
